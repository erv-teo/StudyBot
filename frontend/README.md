# StudyBot Frontend Dashboard

A modern React + TypeScript dashboard for managing the StudyBot RAG system, built with Vite, Mantine UI, and modular component structure.

## Features

- **Real-time Status Monitoring**: View bot health and backend status
- **Document Management**: Browse and search documents, view chunk details
- **Chunk Operations**: View, edit, and delete individual chunks
- **Modern UI**: Built with Mantine components and theming
- **Responsive Design**: Optimized for desktop and mobile

## Prerequisites

- Node.js 18+
- npm (recommended)
- StudyBot backend running at `http://localhost:8000`

## Installation

1. Install dependencies:

   ```bash
   npm install
   ```

2. Start the development server:

   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.
